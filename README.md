# 我的天气
根据第一行代码仿写，加入自己想加入的功能  
具体功能有：  
        基本的天气预报  
        加入前台天气的关闭  
        加入开启和停止后台更新  
用到的第三方依赖库有:okhttp,gson,glide,litepal
